mcap.message_queue module¶

class mcap.message_queue.MessageQueue(log_time_order: bool, reverse: bool = False)[source]¶

Bases: object

A queue of MCAP messages and chunk indices.

Parameters:

  • log_time_order – if True, this queue acts as a priority queue, ordered by log time. if False, pop() returns elements in insert order.

  • reverse – if True, order elements in descending log time order rather than ascending.
